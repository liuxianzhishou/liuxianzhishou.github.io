                # 🕐 Java 类

                # 访问权限

                • private : 只能被同一包下的同一类访问
                • default ： 只能被同一包下的类访问
                • protected : + 能被不同包下的子类访问
                • public : 所有包下的类均可访问

                # 构造函数

                # 调用时间

                • 在新对象实例化时被调用

                # 特点

                1. 方法名与类名相同
                2. 无返回值
                3. 可以重载，不可子类重写

                # 注

                • 在没有自定义引入类的构造方法时，类编译时自动生成一个无参构造函数；
                • 当有构造方法时，无论是否有参，均不会自动生成无参构造函数。

                # 详细说明：

                • static :

                  • 作用范围： 属性 & 方法 & 代码块 & 类
                  • 作用于属性： 类静态变量，不属于类的实例，可以在类未实例化时调用，通过 类名.方法名 的形式
                  • 作用于方法： 静态方法不属于类的实例，只能使用静态变量和调用静态方法
                  • 作用于代码块： 常用于初始化类静态变量，只能使用静态变量，只在类加载时载入内存，执行一次
                  • 作用于类： 作为静态内部类，此时相当于一个外部类，且只能调用外部类的静态操作

                • final ：

                  • 作用范围： 属性 & 方法 & 类
                  • 作用于属性： 属性值一旦被赋值，则不可变
                  • 作用于方法： 不可被重写
                  • 作用于类： 不可被继承

                • abstract ：

                  • 作用范围： 方法 & 类
                  • 作用于方法： 抽象方法，不含具体实现，所在的类一定为 抽象类
                  • 作用于类： 抽象类，不能被实例化，只能被继承，不一定含有抽象方法
